Feature #3

Dispatching Optimization

The Hungarian algorithm solves the optimal assignment problem: which employee should work on which product for each machine to minimize total production time.

HOW IT WORKS

Historical Analysis
Query avg task duration per (employee, machine, product) — excluding anomalous logs
Cost Matrix
Build NxN cost matrix where cost[i][j] = expected time for employee i on product j
Hungarian Algorithm
scipy.optimize.linear_sum_assignment finds global optimum in O(n³)
Optimal Assignments
Compare actual vs. optimal — quantify exact improvement potential
